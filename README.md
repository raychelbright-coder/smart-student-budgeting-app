# Smart Student Budgeting App

Full-stack budgeting application that extracts receipt data using OCR and generates personalised financial insights using GPT-4o mini.

## Technologies
- React
- Flask
- SQLite
- OCR
- OpenAI API

## Features
- Receipt scanning
- Spending analytics
- Merchant comparisons
- AI-generated budgeting advice

## Purpose
Built as a final-year honours project focused on AI-assisted financial management for students.
